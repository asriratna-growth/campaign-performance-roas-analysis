# Documentation

## Project Overview

This project analyzes marketing campaign performance across multiple channels using SQL, Google BigQuery, and Looker Studio.

The analysis evaluates:

- Marketing Spend
- Revenue Generation
- Return on Advertising Spend (ROAS)
- Campaign Funnel Performance
- Conversion Efficiency
- Customer Acquisition Cost (CPA)

## Business Objective

Identify the most efficient marketing channels and evaluate campaign profitability by measuring spend, revenue, conversion performance, and acquisition efficiency.

## Tools Used

- SQL
- Google BigQuery
- Looker Studio

## Dataset

Marketing Campaign Performance Dataset Across Multiple Channels (2025)

## Dashboard Pages

### 1. Campaign Performance Overview

- Total Spend
- Total Revenue
- Overall ROAS
- Channel Revenue Analysis
- Channel Spend Analysis
- ROAS by Channel

### 2. Campaign Funnel & Conversion Analysis

- Marketing Funnel Performance
- CTR Analysis
- Lead Rate Analysis
- Lead-to-Conversion Analysis
- Customer Acquisition Cost (CPA)

## Key Findings

- Total marketing spend reached $25.5M
- Total revenue generated exceeded $51M
- Overall ROAS achieved 2.0x
- Campaigns generated more than 10M conversions
- Search delivered competitive ROAS with lower spend requirements
